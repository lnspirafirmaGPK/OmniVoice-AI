# Contributing to OmniVoice AI

Thank you for contributing.

## Development Setup

Install dependencies:

npm install

Run development server:

npm run dev

## Coding Guidelines

Follow:

- ESLint rules
- Prettier formatting
- AGENTS.md design guidelines

## Pull Request Process

Before submitting a PR:

- Run lint
- Run tests
- Update documentation
- Ensure CI passes

## Commit Convention

Recommended format:

type(scope): description

Examples:

feat(voice): add order status intent
fix(api): correct webhook authentication
docs(readme): update setup instructions
